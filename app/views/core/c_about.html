About

Features

  • Mysql, Express, Angular, Node and Redis(js)
  • Socket.io
  • JSON web tokens
  • Sequelize ORM - because relations and transactions matter
  • Mailgun with Node Mailer
  • Stripe
  • Open Graph Protocol with automatic meta tag updates on each view
  • SEO in mind. Google renders hashbangs
  • Mobile support. Uses FastClick and a loading bar
  • Angulartics - web analytics in your views
  • Scheduler - cron style
  • Minigun testing
  • Animations using animate.css
  • Bootstrap & bootstrap-ui
  • Forms - validation popovers, informative buttons
  • Fonts! - Google, IonIcons, Font Awesome, Bootstrap glyphs
  • Uses Grunt - based on angular Yeoman generator

Authentication

  • Quick and easy user sign up
  • Confirm accounts via email
  • Reset password via email or via Admin dashboard
  • Social logins
  • JWT refresh tokens
  • JWT 'session' tokens - will not refresh after x hours
  • Persistent JWT tokens for use with mobile apps - sign in once only
  • JWT tokens are revoked on the server on sign out
  • ACL - assign roles with different levels - admin, members, everyone, anonymous
  • Reset password to revoke ALL tokens
  • Use JWT with Socket.io

Security

  • JSON Web Token - no cookies
  • Strict JSON schema validation
  • Intrusion Detection System - detect & prevent malicious activity
  • Rate limit endpoints
  • Too busy - keep serving when your app gets massively popular

Audit Logs

  • Searchable logs with nvD3 charts
  • Log particular routes/actions - 200 codes
  • Log security - 400 codes
  • Log server errors - 500 codes
  • Log client errors - JSNlog
  • Geographic, CPU, lag, memory data
  • Common Log Format support

Structure

  • Detailed route viewer
  • Route tester - Minigun with jsonschema faker
  • Global config and library
  • Global error handler
  • Consistent code
